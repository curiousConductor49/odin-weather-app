# webpack-project-template

This is a quick template you can update and use for a project made in Webpack. :)
To use:

- select this as a template when making a new repo
- locally clone your repo
- run `npm install` to install all the necessary packages
- update the corresponding fields in `package.json` and `package-lock.json` with your project's name, GitHub url, and GitHub homepage
- push your changes
- code away!
